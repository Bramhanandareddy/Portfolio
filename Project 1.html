Basics of REST API in Frontend

This project explores the fundamentals of using REST API in frontend development. It demonstrates how to make asynchronous API calls, handle responses, and update the UI dynamically.

Technologies used in this project include HTML, CSS, JavaScript, and Fetch API to interact with the RESTful backend.

The project is designed to create a simple web application that interacts with a public REST API to retrieve and display data in the frontend. The application showcases how to handle data asynchronously, display loading indicators, and handle errors during API calls.

You can find the source code for this project on my GitHub repository: GitHub Repository.
